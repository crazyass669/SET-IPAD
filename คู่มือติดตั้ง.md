# คู่มือ SET Dashboard

## สิ่งที่ต้องมี

- Windows 10 / 11
- Python 3.10 ขึ้นไป
- Internet (สำหรับดึงข้อมูลหุ้น)

---

## ติดตั้งครั้งแรก

### ขั้นตอนที่ 1 — ติดตั้ง Python

1. เปิดเว็บ [python.org/downloads](https://www.python.org/downloads/)
2. กด **Download Python 3.x.x**
3. เปิดไฟล์ที่โหลดมา
4. **สำคัญ** — ติ๊กถูกที่ **"Add Python to PATH"** ก่อนกด Install
5. กด **Install Now** และรอจนเสร็จ

### ขั้นตอนที่ 2 — วางไฟล์โปรแกรม

คัดลอกโฟลเดอร์ **SET_Dashboard** ไปไว้ที่ไหนก็ได้ เช่น `C:\SET_Dashboard\`

ตรวจสอบว่ามีไฟล์เหล่านี้ครบ:

```
SET_Dashboard/
├── start.bat
├── app.py
├── set_data_fetcher.py
├── set_dashboard.html
├── listedCompanies_en_US.xlsx
└── requirements.txt
```

### ขั้นตอนที่ 3 — เปิดโปรแกรม

ดับเบิ้ลคลิกที่ **`start.bat`**

- ครั้งแรก: โปรแกรมจะติดตั้ง library อัตโนมัติ รอสักครู่
- Browser จะเปิดขึ้นมาที่ `http://localhost:5001`
- หน้าต่างสีดำ (Terminal) ต้องเปิดทิ้งไว้ตลอดเวลาที่ใช้งาน

### ขั้นตอนที่ 4 — ดึงข้อมูลครั้งแรก

1. กดปุ่ม **⟳ Full Refresh** มุมบนขวา
2. รอประมาณ **7–10 นาที** (ดึง price history + Fundamentals หุ้นกว่า 900 ตัว + ดัชนีกลุ่ม 30+ ดัชนี + DR/DRx 80+ ตัว)
3. เมื่อเสร็จ ข้อมูลจะแสดงครบทุกหน้า

## หน้าต่างๆ ในโปรแกรม

### 1. Overview
ภาพรวมตลาด SET ประกอบด้วย:
- **Fear & Greed Index** — วัดอารมณ์ตลาด (0 = กลัวมาก, 100 = โลภมาก)
- **Market Regime** — Bull / Neutral / Bear score คำนวณจาก % หุ้นเหนือ EMA200/EMA50 + RS80 + Positive 1M (0–100)
- **Market Breadth** — % หุ้นที่อยู่เหนือ EMA50/200, % หุ้นที่ขึ้น/ลง
- **Top Sectors** — Sector ที่แข็งแกร่งที่สุด (1M%)
- **Top/Bottom หุ้น** — หุ้นที่ขึ้น/ลงมากที่สุดวันนี้
- **52W New High Chart** — กราฟจำนวนหุ้นที่ทำ 52W High ใหม่รายวัน (ย้อนหลัง 63 วัน)

---

### 2. Rotation
แผนที่ Sector/Industry Rotation แกน X = 3M Return, แกน Y = 1M Return

| Quadrant | ความหมาย |
|---|---|
| ขวาบน (เขียว) | Leading — Trend ดี + Momentum แรง |
| ซ้ายบน (น้ำเงิน) | Improving — กำลัง Recover |
| ขวาล่าง (เหลือง) | Weakening — Trend ดีแต่ Momentum ชะลอ |
| ซ้ายล่าง (แดง) | Lagging — Trend แย่และอ่อนแรง |

- **ลูกศรบน Bubble** — ชี้ทิศทางที่ Sector กำลังเคลื่อนที่ (จากตำแหน่งก่อนหน้าไปยังปัจจุบัน)
- **Hover บน Bubble** — แสดงเส้น trail ย้อนหลัง (1Y→6M→3M→ปัจจุบัน)
- **ตัวเลขในวงกลม** — RS Score เฉลี่ยของหุ้นใน Sector นั้น

---

### 3. ดัชนีกลุ่มอุตสาหกรรม SET & mai
ดัชนีกลุ่ม 30+ ดัชนี พร้อม RS Score, Momentum และสัญญาณกลับตัว

**ค่าที่แสดงในการ์ดแต่ละใบ:**

| ค่า | ความหมาย |
|---|---|
| RS SET | Relative Strength เทียบหุ้น 930 ตัวใน SET (0–99) |
| Momentum | `1M% − (6M% ÷ 6)` — วัดว่า momentum กำลังเร่งขึ้นหรือไม่ |
| RS Trend | ↑ RS ดีขึ้น / → ทรงตัว / ↓ อ่อนลง (เทียบ 4 สัปดาห์ก่อน) |
| 🔄 กลับตัว | badge แสดงเมื่อ: 1M%>0 + 6M%<15 + Momentum>1 |

**ปุ่มเรียงลำดับ:** 1D% · 1W% · 1M% · 3M% · 6M% · 1Y% · RS SET · Momentum · 🔄 กลับตัว

**วิธีหา Sector ต้นรอบ (กด 💡 วิธีดู Sector ต้นรอบ เพื่อดูคำแนะนำ):**
1. กดปุ่ม **🔄 กลับตัว** — กรองเฉพาะ Sector ที่ผ่านเกณฑ์
2. กดปุ่ม **Momentum** — เรียงจากมากไปน้อย
3. คลิกการ์ดที่มี RS Trend ↑ → แท็บ **📋 หุ้นในกลุ่ม** เพื่อดูหุ้นรายตัว

**แท็บในกราฟ Modal:**
- **📈 กราฟ** — กราฟราคาดัชนีพร้อม EMA
- **📋 หุ้นในกลุ่ม** — รายชื่อหุ้นทุกตัวใน Sector นั้น คลิกดูกราฟได้ทันที
- **🔗 TradingView** — เปิดดัชนีนั้นใน TradingView

---

### 4. RS 80+ Leaders
หุ้นที่มี RS Score ≥ 80 — แข็งแกร่งกว่า 80% ของตลาด มี 2 แท็บ:
- **🏆 RS 80+ Leaders** — หุ้น Top 20% ของตลาด
- **⚡ Emerging Leaders** — หุ้น RS 35–79 ที่ momentum กำลังพุ่ง (1M% ≥ 3% + เหนือ EMA50)

**RS Score** คืออะไร:
- วัดความแข็งแกร่งเทียบกับตลาด (0–99)
- คำนวณจาก: `(1M×2 + 3M×1 + 6M×1 + 1Y×1) / 5` แล้ว rank เป็น percentile เทียบหุ้น ~930 ตัว
- ≥ 80 = Top 20% ของตลาด

---

### 5. Sectors & Industries
ผลตอบแทนและ RS เฉลี่ยแต่ละหมวดธุรกิจ

คอลัมน์สำคัญ:
- **% >EMA50** — สัดส่วนหุ้นใน Sector ที่อยู่เหนือ EMA50
- **New High** — % หุ้นที่ราคา ≥ 52W High (ทำจุดสูงสุดใหม่)
- คลิก Sector/Industry เพื่อดูรายชื่อหุ้นในหมวดนั้น

---

### 6. 🎯 Stage Analysis (Weinstein Stages)
จัดหุ้นทุกตัวใน SET เป็น 4 Stage ตามแนวคิด Stan Weinstein:

| Stage | ความหมาย | เงื่อนไข |
|---|---|---|
| 2 — Advancing | โซนซื้อ — ราคาขึ้น + momentum | เหนือ EMA200 + slope ≥ +1.5% |
| 1 — Basing | ฐานสะสม — รอ Breakout | ใต้ EMA200 + slope ≥ −1.5% |
| 3 — Topping | ยอดเขา — ระวัง | เหนือ EMA200 + slope < +1.5% |
| 4 — Declining | ขาลง — หลีกเลี่ยง | ใต้ EMA200 + slope < −1.5% |

**คอลัมน์พิเศษ:** RS, Slope% (ความชัน EMA200), ATR% (ความผันผวนรายวัน), 1M%, 3M%

### 7. หุ้นทั้งหมด
ตารางหุ้นทั้งหมดใน SET + mai กรองและ sort ได้

- **ค้นหา** — พิมพ์ Symbol เช่น `KBANK`, `AOT`
- คลิก Symbol เพื่อดู **Chart Modal** (กราฟพร้อม EMA20/50/200 และ ATH line)
- กดไอคอน **↗** ข้างชื่อหุ้น เพื่อเปิดใน TradingView Daily โดยตรง
- คอลัมน์ **ATR%** — ความผันผวนรายวันเฉลี่ย 14 วัน ใช้คำนวณ position size
- คอลัมน์ **Stage** — Weinstein Stage 1–4 ของหุ้นแต่ละตัว
- เลือก Timeframe: 1M · 3M · 6M · 1Y · 5Y · Max

### 8. Stock Screener
กรองหุ้นตามเงื่อนไขที่ต้องการ (กด **💡 วิธีใช้ Screener สำหรับมือใหม่** เพื่อดูคำแนะนำ)

**Filter ที่มี:**

| Filter | ความหมาย |

| RS Score ≥ | ความแข็งแกร่งขั้นต่ำ (0–99) |

| ห่าง High ≤ (%) | ห่างจาก 52W High ไม่เกิน X% |

| ห่าง Low ≤ (%) | ห่างจาก 52W Low ไม่เกิน X% |

| Div Yield ≥ (%) | อัตราเงินปันผลขั้นต่ำ |

| Above EMA20/50/200 | ราคาอยู่เหนือ EMA ที่เลือก |
| Golden Cross | EMA50 > EMA200 (Bull Market State) |
| ตลาด | กรอง SET หรือ mai |
| กลุ่ม | กรองตาม Industry Group |

**Preset แนะนำ (เรียงตามความเหมาะสมสำหรับมือใหม่):**

| # | Preset | เงื่อนไข | เหมาะกับ |
|---|---|---|---|
| 1 | 🔄 ต้นรอบ | RS≥50, 1M%≥3, EMA50✓ | หุ้นที่เพิ่งเริ่มแข็งแกร่ง — เหมาะที่สุดสำหรับเริ่มต้น |
| 2 | ⚡ Strong Momentum | RS≥70, 1M%≥5, 3M%≥10, EMA50+200✓ | ตลาดขาขึ้น หุ้นแข็งแกร่งทุกด้าน |
| 3 | 📈 Breakout | RS≥60, RVOL≥150%, EMA50✓ | หาจังหวะ entry วัน Volume พุ่ง |
| 4 | 🎯 Near ATH | ห่าง ATH≤10%, RS≥50 | หุ้นใกล้ทำ All-Time High ใหม่ |
| 5 | 💰 High Yield | Div≥3%, RS≥60 | Income portfolio ที่ยังมี momentum |
| 6 | 💎 Value Turnaround | PBV≤1.5, RS≥45, 1M%≥0 | หุ้น Undervalued เริ่มกลับตัว |

### 9. 52W High Watch
หุ้นที่ราคาใกล้จุดสูงสุด 52 สัปดาห์

- **Top 5%** = ห่าง High ไม่เกิน 5% (High Tight Zone)
- **NEW HIGH** = ทำราคาสูงสุดใหม่ (price ≥ 52W High)
- **Range Position** = ตำแหน่งราคาใน 52W Range (100% = อยู่ที่ High, 0% = อยู่ที่ Low)

---

### 10. 52W Low Watch
หุ้นที่ราคาใกล้จุดต่ำสุด 52 สัปดาห์ — ใช้มองหาสัญญาณ Turnaround

สัญญาณกลับตัวเบื้องต้น: RS เริ่มดีขึ้น + 1D%/1W% กลับมาเป็นบวก

---

### 11. Breakout Radar
คัดหุ้นที่มีครบ 3 เงื่อนไขพร้อมกัน:
1. **RS สูง** — แข็งแกร่งกว่าตลาด
2. **ใกล้ 52W High** — พร้อม Breakout
3. **เหนือ EMA** — อยู่ใน Uptrend

ปรับ threshold RS, ระยะห่าง High, และ EMA filter ได้

---

### 12. Momentum Alignment
หุ้นที่ momentum เป็นบวกพร้อมกันหลาย Timeframe

| Filter | เงื่อนไข |
|---|---|
| ทุก TF | 1D > 0% · 1W > 0% · 1M > 0% · 3M > 0% |
| 1W·1M·3M | ตัด 1D ออก เน้น trend ระยะกลาง |
| 1M·3M | ผ่อนคลายที่สุด universe กว้าง |

### 13. Heatmap
แผนที่สีแสดงทุกหุ้น จัดกลุ่มตาม Sector — ทุกกล่องขนาดเท่ากัน สีเท่านั้นที่ต่างกัน

เลือก metric ที่ต้องการแสดงด้วยสี:
1D% · 1W% · 1M% · 3M% · YTD% · 1Y% · RS Score · Vol Ratio · % from 52W High · ATH Distance · Momentum

---

### 14. EMA Breadth
แสดง % หุ้นที่อยู่เหนือ EMA แต่ละเส้นในแต่ละ Sector

- **% > EMA20** — trend ระยะสั้น (~1 เดือน)
- **% > EMA50** — trend ระยะกลาง (~2.5 เดือน)
- **% > EMA200** — trend ระยะยาว (~10 เดือน)
- **Trend Score** — ค่าเฉลี่ยของทั้งสามเส้น: `(EMA20% + EMA50% + EMA200%) ÷ 3` (0–100)

### 15. Fundamentals

Preset กรอง: High Yield ≥ 3% · P/BV < 1 · P/E < 15 · มีข้อมูลครบ

- แหล่งข้อมูล: mrlikestock.com (ดึงเมื่อค้นหาแต่ละตัว)

### 17. Watchlist

### 18. DR/DRx
ราคาและ Performance ของ Underlying Stocks ที่มี DR/DRx เทรดบน SET

- ครอบคลุมหุ้นต่างประเทศจาก US, HK/CN, Japan, Europe, Singapore, Vietnam, Taiwan
- กรองตามภูมิภาค หรือค้นหาด้วย Symbol / ชื่อบริษัท
- มี 2 มุมมอง: **Table** (ข้อมูลครบ) และ **Cards** (กราฟ trend 100 วัน)
- กดไอคอน **↗** เพื่อเปิดใน TradingView (ใช้ exchange prefix ที่ถูกต้องตามภูมิภาค)
- ข้อมูลอัปเดตโดย **⚡ Quick Update** (เฉพาะวันที่ขาดหาย) หรือ **⟳ Full Refresh**

## Chart Modal (กราฟหุ้นแต่ละตัว)

คลิก Symbol ในหน้าใดก็ได้เพื่อเปิดกราฟ หรือกดไอคอน **↗** เพื่อข้ามไป TradingView โดยตรง

**แท็บในหัว Modal:**
- **📈 กราฟ** — กราฟราคาพร้อม EMA
- **🧾 งบการเงิน** — รายได้ กำไร กระแสเงินสด (ดึงจาก Yahoo Finance)
- **TradingView** — เปิดหุ้นตัวนั้นใน TradingView (tab ใหม่)

**Timeframe:** 1M · 3M · 6M · 1Y · 5Y · Max

**เส้นในกราฟ:**
- เส้นน้ำเงิน = ราคา
- เส้นม่วง = EMA20
- เส้นส้ม = EMA50
- เส้นแดง = EMA200
- เส้นประม่วง = ATH (All-Time High จากประวัติทั้งหมด)

**Metrics แสดงด้านบน:** ราคา · 1D% · 1W% · 1M% · 3M% · YTD% · 52W High/Low · ATH · % จาก ATH · P/E · P/BV · Div Yield

## แก้ปัญหาเบื้องต้น

**เปิด `start.bat` แล้วปิดเองทันที**
- เปิด Command Prompt แล้วพิมพ์ `python --version`
- ถ้าขึ้น error แสดงว่า Python ไม่ได้ Add to PATH → ติดตั้งใหม่

**Browser ขึ้น "This site can't be reached"**
- รอ 5 วินาทีแล้วกด Refresh ใน browser
- ตรวจว่าหน้าต่างสีดำยังเปิดอยู่
- ลองพิมพ์ `localhost:5001` ในแถบ address

**กด Refresh แล้วข้อมูลไม่อัปเดต**
- กด **Ctrl+Shift+R** ใน browser เพื่อ hard refresh
- ถ้ายังไม่อัปเดต ปิด Terminal แล้วเปิด `start.bat` ใหม่

**ต้องการอัปเดตรายชื่อหุ้น**
- โหลดไฟล์ใหม่จาก [set.or.th](https://www.set.or.th/th/market/product/stock/quote/all-stock)
- แทนที่ `listedCompanies_en_US.xlsx` แล้วกด Full Refresh

**Yahoo Finance rate limit (ข้อมูลไม่มา / error)**
- รอ 15–30 นาทีแล้วลองใหม่
- ข้อมูลเดิมจะยังแสดงอยู่ระหว่างรอ

---

## ข้อมูลที่ใช้และความถี่

| ข้อมูล | แหล่ง | อัปเดตเมื่อ |
|---|---|---|
| ราคาหุ้น SET/mai | Yahoo Finance (.BK) | กด Quick Update / Full Refresh |
| ดัชนีกลุ่มอุตสาหกรรม | Yahoo Finance (.BK) | กด Quick Update / Full Refresh |
| DR/DRx Underlying | Yahoo Finance (symbol ต้นฉบับ) | กด Quick Update / Full Refresh |
| P/E · P/BV · Div Yield · Market Cap | Yahoo Finance | กด Full Refresh เท่านั้น |
| งบการเงิน (รายได้/กำไร/กระแสเงินสด) | Yahoo Finance | ดึงเมื่อเปิด Modal งบการเงิน |
| Valuation Band (P/E Band, P/BV Band) | mrlikestock.com | ดึงเมื่อค้นหาแต่ละตัว |
| รายชื่อหุ้น SET | listedCompanies_en_US.xlsx | อัปเดตไฟล์ manual |
