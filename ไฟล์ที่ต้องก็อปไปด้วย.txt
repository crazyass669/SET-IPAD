====================================================
  SET Dashboard — ไฟล์ที่ต้องก็อปไปด้วยเวลาย้ายเครื่อง
====================================================

[ จำเป็นต้องมี — ก็อปทุกไฟล์นี้ ]

  start.bat                    ← ดับเบิ้ลคลิกเปิดโปรแกรม
  app.py                       ← Flask Web Server (หัวใจหลัก)
  set_data_fetcher.py          ← โค้ดดาวน์โหลดข้อมูลจาก Yahoo Finance
  set_dashboard.html           ← หน้าเว็บ Dashboard ทั้งหมด
  requirements.txt             ← รายการ Library ที่ต้องติดตั้ง
  listedCompanies_en_US.xlsx   ← รายชื่อหุ้น SET (ดาวน์โหลดจาก set.or.th)

----------------------------------------------------

[ ข้อมูลที่ดาวน์โหลดแล้ว — ก็อปด้วยถ้าไม่อยากรอ Full Refresh อีกครั้ง ]

  set_data.json        ← ข้อมูลหุ้นทุกตัว (ราคา, RS, EMA, Fundamentals) ~16 MB
  set_history.json     ← ประวัติราคาย้อนหลังทุกตัว (ใช้กราฟ + EMA200) ~100 MB
  dr_cache.json        ← ข้อมูล DR/DRx ต่างประเทศ ~32 MB
  indices_cache.json   ← ข้อมูลดัชนีกลุ่ม 30+ ดัชนี ~4 MB

  ** ถ้าไม่ก็อปไฟล์ข้อมูล → ต้อง Full Refresh ใหม่ในเครื่องใหม่ (~1-2 ชั่วโมง)

----------------------------------------------------

[ คู่มือ — ก็อปไว้อ่านที่เครื่องใหม่ (ไม่จำเป็นสำหรับการรันโปรแกรม) ]

  คู่มือการใช้งาน.html    ← เปิดด้วย Browser อ่านได้เลย ไม่ต้องรัน Server
  คู่มือติดตั้ง.md        ← วิธีติดตั้งตั้งแต่ต้น (Markdown)

----------------------------------------------------

[ ไม่ต้องก็อป — สร้างเองอัตโนมัติ ]

  __pycache__/            ← Python cache (สร้างใหม่เองตอนรัน)
  *.bak                   ← ไฟล์ backup เก่า
  response.txt            ← log ชั่วคราว
  server_out.txt          ← log ชั่วคราว
  server_err.txt          ← log ชั่วคราว
  test_batch.py           ← script test ภายใน
  test_syms.py            ← script test ภายใน
  ttm_test_results.json   ← ผลลัพธ์ test ภายใน

====================================================

[ ขั้นตอนหลังก็อปไปเครื่องใหม่ ]

  1. วางทั้งโฟลเดอร์ไว้ที่ไหนก็ได้ในเครื่องใหม่
  2. ตรวจสอบว่าติดตั้ง Python 3.10+ แล้ว
     → โหลดที่ python.org (ติ๊ก "Add to PATH" ตอนติดตั้ง)
  3. ดับเบิ้ลคลิก start.bat
     → จะติดตั้ง Library อัตโนมัติครั้งแรก (flask, yfinance, pandas ฯลฯ)
  4. Browser เปิดขึ้นเองที่ http://localhost:5001
  5. ถ้าไม่ได้ก็อปไฟล์ข้อมูล → กด Full Refresh เพื่อดาวน์โหลดใหม่
